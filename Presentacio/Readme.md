# Presentación – Programació Multiplataforma i Distribuïda

Esta carpeta contiene la presentación realizada para la asignatura **Programació Multiplataforma i Distribuïda**.

Incluye:

- Una demo desarrollada con **Flask**.
- Creación de una **API** funcional.
- Explicación detallada de todos los conceptos y pasos realizados.

El objetivo de esta presentación es mostrar de manera clara y práctica cómo construir aplicaciones distribuidas y APIs con Flask.
