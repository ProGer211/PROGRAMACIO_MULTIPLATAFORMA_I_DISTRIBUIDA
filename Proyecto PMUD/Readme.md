# Proyecto PMUD – Programació Multiplataforma i Distribuïda

Esta carpeta contiene el **proyecto principal** de la asignatura.

Desarrollos incluidos:

- Página web completa con frontend y backend.
- **API** para consultar información sobre fútbol.
- **Base de datos** que almacena ligas, países, equipos y estadísticas.

El proyecto permite navegar y consultar datos de fútbol de manera interactiva, mostrando los conocimientos adquiridos en programación multiplataforma y desarrollo de aplicaciones distribuidas.
