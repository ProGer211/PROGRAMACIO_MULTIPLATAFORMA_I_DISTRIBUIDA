# Exercicis – Programació Multiplataforma i Distribuïda

Esta carpeta contiene todos los **ejercicios realizados en clase** para la asignatura.

Incluye prácticas para:

- Reforzar conceptos de programación distribuida.
- Trabajar con APIs y bases de datos.
- Desarrollar proyectos multiplataforma.

Los ejercicios sirven como complemento práctico a los proyectos principales del curso.
